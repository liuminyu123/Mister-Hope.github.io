    # 秋之思【十四】

    肩上蝶，袖底清风，如烟归去，散落天涯。

    ♢

    碧波亭，繁花似锦，翘首天涯何为暖?

    羽翼化蝶，衣袂飘飘，怎能红尘恋?

    ♢

    惜别，痴念，月残上弦!
